⚡ DEPLOY IN 3 STEPS:

1️⃣  PUSH TO GITHUB
   git add .
   git commit -m "Production ready - contact form complete"
   git push origin main

2️⃣  ADD ENV VARS IN NETLIFY
   Go to: Site Settings → Environment variables
   
   Required (2):
   • SUPABASE_URL = [from supabase.com]
   • SUPABASE_SERVICE_ROLE_KEY = [from supabase.com]
   
   Optional (for emails):
   • RESEND_API_KEY = [from resend.com]

3️⃣  DEPLOY
   • Automatic: Push triggers deploy (wait 2-3 min)
   • Manual: Deploys → Trigger deploy
